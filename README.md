### Walmart Satış Data Analitiqi Layihəm
Böyük data kütlələrini idarəçilər üçün anlaşıqlı biznes qərarlarına çevirmək pərakəndə (retail) sektorunda ən kritik amildir. Mən də Walmart-ın tarixi satış datalarını analiz edərək dynamic və interaktiv bir Power BI Dashboard-u hazırladım.

Layihə 3 əsas analitik səhifədən ibarətdir:

📈 Satış və Zaman Trendləri: İllik və aylıq satış dinamikasını çıxardım, reytinq və Top 10 mağazanın performansını analiz etdim.

🌍 Xarici Faktorların Təsiri: Hava temperaturu, yanacaq qiymətləri, inflyasiya (CPI) və işsizlik faizinin satışlarla korolyasiyasını Scatter Chart və trend xətləri ilə araşdırdım.

🎉 Bayram və Mövsümlülük: Xüsusi DAX formulları yazaraq bayram günləri ilə adi günlərin ümumi satış payını (Bayram_Satislari və Adi_Gun_Satislari) hesabatda dinamikləşdirdim. Mağazaların aylıq performansını isə Matrix Heatmap cədvəlinə çevirdim.

🛠️ Texniki Alətlər: Power BI, Power Query, DAX (Calculated Measures), Data Modeling.
